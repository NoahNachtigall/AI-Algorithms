# Tic Tac Toe with AI

This Python script implements a Tic Tac Toe game with an unbeatable AI opponent using the Minimax algorithm. The AI will never lose.

## Features
- Human vs AI gameplay
- AI uses Minimax for optimal moves
- Simple graphical interface with Pygame
- Press R to restart the game

## Requirements
- Python 3.x
- Pygame
- NumPy

## Installation
1. Install dependencies:
   ```
   pip install pygame numpy
   ```

## Usage
Run the script:
```
python tic_tac_toe.py
```
- Click on the grid to make your move (O)
- AI plays as X
- The game detects wins, losses, and draws

## Algorithm Details
The Minimax algorithm evaluates all possible game states to choose the best move, assuming optimal play from both sides.
